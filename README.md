# 🐚 Minishell

Mini reproduction d’un shell Unix réalisée dans le cadre de 42 Perpignan.

## 🚀 Features
- Exécution de commandes  
- Pipes (`|`) et redirections (`>`, `<`, `>>`)  
- Variables d’environnement  
- Builtins : `cd`, `echo`, `pwd`, `export`, `unset`, `env`, `exit`  
- Gestion des signaux
